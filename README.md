# Product Entry System

This is a simple Product Entry System project where users can add products with their category, price, and quantity.

I have built a form with four inputs:
- Product Name  
- Category  
- Price  
- Quantity  

When a product is added, it is displayed in a table below. Each record includes **Update** and **Delete** buttons.

Users can:
- Add new products
- View products in a table format
- Update existing product details
- Delete products from the list
